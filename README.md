# avr-tools-build
Automate build and install of AVR compiler and toolchain

## build-avr.sh
This is a script for compiling the avr toolchain in a local directory.
At default this will download the source code for the following packages:
	Binutils	2.26
	GCC		5.3.0
	GMP		6.1.0
	MPC		1.0.3
	ISL		0.16.1
	AVR libc	2.0.0
	GDB		7.11
	simAVR		1.3
